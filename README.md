# Queen's Attack | Professional Logical Engine

![Ataque Reina Banner](ataque_reina_banner.png)

## Overview
An optimized Java implementation of the Queen's Attack II challenge. This project demonstrates high-performance algorithmic logic combined with a premium CLI experience.

## 🚀 Key Features
- **O(k) Performance**: Efficiently calculates attackable squares in 8 directions in linear time relative to the number of obstacles.
- **Robust Logic**: Comprehensive handling of board boundaries and null input protection.
- **Executive UI**: Clean, ANSI-colored console interface with professional ASCII branding.

## 🛠️ Technical Stack
- **Language**: Java 21
- **Build System**: Maven
- **Testing**: JUnit 5

## 📖 How to Run
Ensure you have Maven installed, then execute:
```bash
mvn compile exec:java -Dexec.mainClass="ar.edu.utn.frc.tup.piii.App"
```
